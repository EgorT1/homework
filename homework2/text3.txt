git init-Эта команда используется для запуска нового репозитория.
Usage: git init [repository name]
git clone-Эта команда используется для получения репозитория по существующему URL-адресу.
Usage: git clone [url]  
git add-Эта команда добавляет файл в промежуточную область.
Usage: git add [file]  or *
git commit-Эта команда фиксирует любые файлы, которые вы добавили с помощью команды git add, 
а также фиксирует любые файлы, которые вы изменили с тех пор.
Usage: git commit -m “[ Type in the commit message]”
git status-Эта команда выводит список всех файлов, которые должны быть зафиксированы.
Usage: git status
git log-Эта команда используется для просмотра истории версий для текущей ветки.
Usage: git log 
git branch-Эта команда выводит список всех локальных веток в текущем репозитории.
Usage: git branch
git push-его команда отправляет зафиксированные изменения основной ветки в ваш удаленный репозиторий.
Usage: git push [variable name] master
git diff-Эта команда показывает различия в файлах, которые еще не подготовлены.
Usage: git diff
git reset-Эта команда удаляет файл из стадии, но сохраняет содержимое файла.
Usage: git reset [file]
